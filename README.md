# ObserverDDS
网络订阅发布模型  
本程序采用组播方式寻找节点地址  
通过UDP，TCP发送数据  
TCP已经避免粘包；UDP采用最大努力投递  
NetPublisher为发布类，提供类单例；NetSubscriber为订阅类，提供类单例
